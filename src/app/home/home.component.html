Bienvenido a Datos Abiertos

Proyecto hecho por Lucas Vera Toro para explorar la plataforma de Datos abiertos del gobierno de Colombia.

Los datos abiertos son una gran herramienta para involucrarse con la comunidad, además de ser una gran oportunidad para exponer de una manera clara, sencilla y fácil de entender los diferentes datos publicados por entidades públicas o privadas de Colombia. También permite explorar diferentes ideas de negocio y oportunidades de emprendimiento. Ver más (Wikipedia)
